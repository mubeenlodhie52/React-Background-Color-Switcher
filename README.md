# 🎨 React Background Color Switcher

A simple background color switcher built using React and Tailwind CSS.
Clicking the buttons changes the background color dynamically using React state.

## 🚀 Features

Change background color with one click

Built using React useState

Styled with Tailwind CSS

Simple and beginner-friendly project

Responsive UI

## 🖥️ Demo

Buttons available:

🟡 Yellow

🔵 Blue

🟢 Green

Clicking a button updates the page background color instantly.
